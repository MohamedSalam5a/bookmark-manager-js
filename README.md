🔖 Bookmark Manager

A simple and interactive Bookmark Manager web app that allows users to save, visit, and delete website bookmarks .

---

 🚀 Features

- ✅ Add new bookmarks
- 👀 Display all saved bookmarks
- ❌ Delete bookmarks
- 🔗 Visit saved websites directly
- 💾 Save data using Local Storage (CRUD Operations)
- 🛡️ Form Validation:
  - Validate bookmark name
  - Validate URL format
  - Show error messages for invalid inputs

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Bootstrap

⚙️ How It Works

1. Enter Website Name
2. Enter Website URL
3. Click "Submit"
4. Bookmark will be saved and displayed
5. You can Visit or Delete any bookmark
